
## 📌 What is The Prespective?

> **The Prespective** is a next-gen news and discussion platform built for thinkers, readers, and idea-chasers.  
> Real-time news meets curated opinions and constructive debates — all in one seamless experience.

No noise. No bias. Just perspectives. Yours.

---

## ⚡ Highlights

✨ Real-time news from trusted sources  
💬 Open, threaded discussions on every article  
📊 Community-powered upvotes & moderation  
🔎 Topic filters, trending tags & search  
🌗 Sleek dark/light mode UI  
🔐 User login, profiles, and comment history  
🤖 *Coming Soon*: AI-generated summaries, tone detection & bias checks
